# 🥥 Tapiocaria do Dev

Um site moderno e responsivo para uma tapiocaria artesanal, desenvolvido com foco em Mobile First e experiência do usuário. O projeto inclui um cardápio interativo, carrinho de compras funcional e integração com WhatsApp para finalização de pedidos.

![Tapiocaria do Dev](https://images.unsplash.com/photo-1676515231566-eb804ac487c1?q=80&w=1486&auto=format&fit=crop)

## 🚀 Funcionalidades

### 📱 Mobile First & Responsividade
- Design adaptativo que funciona perfeitamente em celulares, tablets e desktops.
- **Menu Hamburguer**: Navegação otimizada para dispositivos móveis.
- **Layouts Flexíveis**: Grid de produtos e seções que se ajustam automaticamente ao tamanho da tela.
- **Tipografia Escalável**: Textos legíveis em qualquer dispositivo.

### 🛒 Carrinho de Compras Completo
- **Adicionar/Remover Itens**: Interface intuitiva para gerenciar o pedido.
- **Controle de Quantidade**: Ajuste fácil da quantidade de cada item.
- **Persistência de Dados**: O carrinho é salvo no `localStorage` do navegador, mantendo os itens mesmo se a página for recarregada.
- **Cálculo Automático**: Subtotais e total geral calculados em tempo real.
- **Badge de Notificação**: Ícone no header mostrando a quantidade de itens no carrinho.

### 💬 Checkout via WhatsApp
- Finalização do pedido envia uma mensagem formatada diretamente para o WhatsApp da loja.
- A mensagem inclui a lista detalhada de itens, quantidades e o valor total.

### 🎨 Design & UI
- **Tema Claro/Escuro**: Alternância de temas com persistência da preferência do usuário.
- **Animações Suaves**: Transições agradáveis no menu, carrinho e botões.
- **Acessibilidade**: Uso de tags semânticas e atributos ARIA para melhor acessibilidade.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e moderna.
- **CSS3**: Estilização avançada com Variáveis CSS (Custom Properties), Flexbox e Grid Layout.
- **JavaScript (Vanilla)**: Lógica do carrinho, manipulação do DOM e interatividade sem dependência de frameworks pesados.

## 📦 Como Executar

1. Clone este repositório:
   ```bash
   git clone https://github.com/seu-usuario/tapiocaria-do-dev.git
   ```
2. Navegue até a pasta do projeto:
   ```bash
   cd tapiocaria-do-dev
   ```
3. Abra o arquivo `index.html` no seu navegador preferido.

## 📝 Estrutura do Projeto

- `index.html`: Estrutura principal e lógica JavaScript inline (para simplicidade neste projeto).
- `styles.css`: Folhas de estilo com design responsivo e temas.
